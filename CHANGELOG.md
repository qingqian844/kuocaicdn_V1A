# 更新日志

## K2.1.4.0 - 2026-07-24

本版本同步授权版最新的 CDN 调度、自建节点和部署能力，并继续按开源版边界移除商业运营功能。

### 新增

- 支持中国大陆、境外和全球三类服务区域分别配置默认 CDN 线路。
- 支持多 CDN 线路组、线路优先级和负载权重分配，可按区域组合不同厂商线路。
- 自建 CDN 增加 DNS 分片和流量均衡，支持按节点状态动态维护解析记录。
- 自建节点增加实时监控、历史监控记录和缓存磁盘管理。
- 自建节点支持 TCP/UDP 端口转发配置，并完善节点 Agent 下发流程。
- 腾讯云 EdgeOne 增加回源 Host 和 Referer 白名单配置。
- 腾讯云 EdgeOne 配置失败时展示云厂商返回的具体失败原因。
- 支持自定义后台访问路径和默认用户头像。
- 增加一键部署、临时管理员密码和首次初始化向导。

### 优化与修复

- 修复域名状态同步后本地状态未及时更新的问题。
- 自建节点 Agent 更新配置时无需重启进程，减少节点服务中断。
- 完善多 CDN 域名创建、区域线路回显和服务区域参数兼容。
- 完善安装中断恢复、外部依赖检查、健康检查失败回滚和备份流程。

### 开源版边界

- 保留全部现有 CDN 厂商、普通用户、余额账户和按量流量计费。
- 系统仅允许存在一个管理员账号，余额仅支持管理员人工调整。
- 保留支付宝实名认证、邮件、短信、微信登录和 DNSPod 配置。
- 移除商业授权校验、在线支付、在线充值和提现功能。
- 移除代理分销、推荐分润、流量包、赠送、礼品和每月福利功能。
- 移除 EdgeOne 付费额度以及厂商账号多账号绑定功能。
- 初始化数据库、部署脚本和发行包均不包含授权文件或真实公私钥。

## K2.1.3.0 - 2026-07-19

本版本将授权版近期的 CDN 管理能力同步到开源版，同时继续执行开源版的功能边界。

### 新增

- 腾讯云 EdgeOne 支持目录、文件后缀和完整路径缓存规则。
- 腾讯云 EdgeOne 支持过滤参数、状态码缓存、响应头规则、URL 鉴权和回源跟随重定向。
- 腾讯云 EdgeOne 支持 Referer、IP、User-Agent、IPv6 和安全防护策略配置。
- 腾讯云 EdgeOne 增加实时流量统计、统计口径修复及创建中断恢复机制。
- 腾讯云 CDN 增加域名归属权验证流程。
- 工单支持图片和普通文件附件。
- 系统 Logo 支持外部链接并同步到控制台。
- 增加自建 CDN 节点、节点组、线路、域名、CNAME、健康检查和故障调度。
- 自建 CDN 支持域名配置、统计、缓存刷新/预热及 TCP/UDP 端口转发。
- 自建 CDN 节点安装支持 CentOS 7，并完善安装并发与超时处理。

### 修复

- 修复腾讯云 EdgeOne 安全规则、缓存规则、访问配置和响应头参数格式问题。
- 修复腾讯云 EdgeOne 缓存规则部署状态误判、回显异常和目录条件表达式问题。
- 修复腾讯云 EdgeOne 流量统计口径偏差及后台全站统计聚合问题。
- 修复阿里云、火山引擎等域名因缺少独立 `domainId` 而被排除在统计之外的问题。
- 云厂商流量展示改用十进制 GB/TB，原始字节值和按量计费数据保持不变。
- 修复后台主题加载白屏、退出登录重复点击、自建 CDN 默认站点和停用状态问题。
- 补充工单详情及附件访问权限校验。

### 开源版边界

- 保留全部现有 CDN 厂商。
- 保留普通用户体系，并限制只能存在一个管理员账号。
- 仅保留按量流量计费、余额自动结算和管理员人工余额调整。
- 移除商业授权校验及商业授权 API 调用。
- 移除支付宝/微信在线收款、在线充值和提现功能。
- 移除流量包购买、赠送、礼品兑换和每月福利功能。
- 移除代理商、分销、代理品牌和推荐分润功能。
- 移除 EdgeOne 根域名付费额度功能。
- 不包含厂商账号多账号绑定和用户绑定功能。
