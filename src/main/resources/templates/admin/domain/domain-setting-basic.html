基本信息

源站配置

主源站指定为 IP 源站时，可配置端口（1~65535），可配置权重（1~100），格式为 IP:端口:权重
热备源站指定为 IP 源站时，可配置端口（1~65535），暂不支持权重配置，格式为 IP:端口

源站需配置为域名源站时，只能填入一个域名；若配置为 IP 源站，多个源站 IP 之间用分号隔开

回源端口：请同步修改回源端口和回源协议类型（回源配置 - 回源方式），避免导致 500 影响业务，HTTPS - 443 HTTP - 80 协议跟随 - 0

源站信息可配置源站域名 - DN、源站 IP - IP 或者 oss 域名 - OSS，可配置端口（HTTP - 80 或 HTTPS - 443），可配置权重（1~100），可配置优先级（主 - A 备 - B）

格式为 源站类型:源站信息:端口:权重:优先级，多个源站之间用分号隔开

注意：HTTP 端口是 80，HTTPS 端口是 443，请同步修改回源端口和回源协议类型（回源配置 - 回源方式）！
OSS 域名必须以 aliyuncs.com 或 aliyun-inc.com 结尾，操作参考文档
