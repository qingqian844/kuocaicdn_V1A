缓存规则

自定义CDN节点指定缓存资源的缓存过期时间规则。支持按照“首页”、“所有文件”、“目录路径”、“文件名后缀”、“全路径”方式指定资源范围。点击添加。
类型 内容 优先级 缓存过期时间 操作
[[${rule.ttl}]]秒 [[${rule.ttl}]]分钟 [[${rule.ttl}]]小时 [[${rule.ttl}]]天
缓存规则新增

新规则仅对后续资源缓存生效，如希望对节点已缓存资源立即生效，请在修改缓存规则后对该部分资源执行刷新操作。
