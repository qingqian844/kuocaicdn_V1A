HTTPS配置

使用加速域名的证书配置HTTPS，并将其部署在全网CDN节点，实现HTTPS安全加速。
ps. 更新域名证书请先关闭 HTTPS 配置保存后，再开启并配置。部分域名已支持快捷配置，请在 HTTPS 快捷配置处保存更新域名证书。

HTTPS 快捷配置

快捷更新加速域名的 HTTPS 证书，更新域名证书信息会开启 HTTPS 访问。
ps. 当前仅支持部分域名，若当前域名不受支持，或者需要查看 HTTPS 状态或者关闭 HTTPS，请使用上面（HTTPS配置）的配置项。

TLS版本配置

TLS（Transport Layer Security）是一种加密协议，用于在两个通信应用程序之间提供保密性和数据完整性。TLS 1.3是最新版本，具有更强的安全性和性能。

强制跳转

将所有HTTP请求强制跳转到HTTPS请求。开启前请确保域名已配置有效证书并开启了HTTPS访问。

301和308为永久重定向，302、303和307为临时重定向。推荐使用302。

HTTP/2

HTTP/2是下一代HTTP协议，相比HTTP/1.1，它提供了更快的加载速度和更低的延迟。

OCSP Stapling

OCSP Stapling是一种用于在线检查SSL/TLS证书状态的机制，可以提高证书验证的效率和隐私性。
