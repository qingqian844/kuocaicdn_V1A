回源方式

请同步修改回源协议类型和回源端口（基本配置 - 源站配置），避免导致 500 影响业务，HTTPS - 443 HTTP - 80 协议跟随 - 0

设置CDN回源的请求方式，选择“协议跟随”时，回源协议遵循客户端的请求方式。

回源URL改写

改写回源URL，提高回源请求URL与源站的匹配度，提升回源命中率，点击添加。

回源URL新增

1、后台有特殊配置的域名暂不支持改写回源URL。
2、通配符“*”可通过“$n”(n=1,2,3...,)捕获。
3、匹配方式为“URL路径”时，待改写回源URL为前缀匹配，且不支持配置参数。

视频拖拽

开启后支持MP4、FLV格式的视频拖拽功能，用户可以自由调整视频播放进度，提升用户体验。

Range回源

开启后回源时只返回指定范围内的内容，能有效缩短大文件的分发时间。

Range回源

需要提交工单配置

Range回源配置需提交工单由技术人员协助配置。如需配置，请联系您的客户经理或提交工单。

回源是否校验ETag

开启此项配置后，当资源在CDN节点缓存过期，用户再次请求该资源时，如果ETag值没有变化，CDN将节点缓存的资源返回给用户，不会触发回源。
